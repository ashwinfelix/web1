About The Project

This portfolio encompasses two captivating web applications: a dynamic Rock-Paper-Scissors game and an immersive landing site featuring stunning parallax scrolling effects. The Rock-Paper-Scissors game, meticulously crafted with HTML, CSS, and JavaScript, offers users an engaging and interactive experience. Its HTML structure provides the foundation for the game layout, while CSS styles elevate the visual appeal with carefully curated design elements and fluid animations. The JavaScript logic drives the game mechanics, enabling users to select their choices and dynamically determine winners. Through seamless integration of front-end technologies, the game delivers a delightful user experience that is both entertaining and intuitive.
In contrast, the landing site exemplifies the power of modern web design techniques, leveraging HTML, CSS, and JavaScript to create an immersive and visually captivating experience. The parallax scrolling effect, a hallmark of the site's design, is achieved through intricate JavaScript manipulation of background elements. As users scroll down the page, background components such as stars, moon, and mountains gracefully shift at varying speeds, imparting a mesmerizing sense of depth and movement. This meticulous attention to detail extends to every aspect of the site, from the sleek typography and vibrant color palette to the seamless navigation and intuitive layout.
Together, these projects showcase the versatility and creativity inherent in web development. They exemplify the fusion of artistry and technology, where code becomes a medium for delivering engaging and immersive digital experiences. Whether engaging in a friendly game of Rock-Paper-Scissors or embarking on a visual journey through a parallax-infused landscape, users are treated to an unforgettable exploration of the possibilities offered by the modern web.

Functionalities

Rock-Paper-Scissors Game:
 User Interaction: Users can engage in a classic game of Rock-Paper-Scissors directly within the browser interface.
Choice Selection: Players can select their choice of Rock, Paper, or Scissors by clicking on the corresponding images.
Dynamic Results: The game dynamically determines the winner between the user and the CPU based on their choices.
Visual Feedback: Visual feedback is provided through animated transitions and result display, enhancing the user experience.
Responsive Design: The game interface is responsive, ensuring compatibility across various devices and screen sizes.
Landing Site with Parallax Scrolling:
Immersive Experience: The landing site offers an immersive experience with visually captivating parallax scrolling effects.
Depth Perception: Background elements such as stars, moon, and mountains move at different speeds, creating a sense of depth and dimensionality.
Visual Appeal: The site boasts sleek typography, vibrant color schemes, and carefully curated design elements to captivate users.
Seamless Navigation: Users can seamlessly navigate through the site, exploring content with intuitive scrolling and interactive elements.
Responsive Design: The site is optimized for responsiveness, ensuring optimal viewing and interaction across a range of devices and screen sizes.
